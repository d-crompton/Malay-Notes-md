## Formal Pronouns

The following pronouns should be used with those you are not familiar with or are expected to be respectful towards

|         |                                             |
| ------- | ------------------------------------------- |
| Saya    | I                                           |
| Saudara | You (Male)                                  |
| Saudari | You (Female)                                |
| Anda    | You (Male or Female)                        |
| Dia     | He, She                                     |
| Kami    | We (_Excluding whoever you are talking to_) |
| Kita    | We (_Including whoever you are talking to_) |

In addition to pronouns, in Malay _any title or name_ can be used to mean _'you'_ when speaking directly to a person. For example, in the sentence **_"Encik perlu teksi?"_** (_'[Does] sir need a taxi?'_) the word **_Encik_** (_sir_) is being used as the word 'you'. Therefore, in context this sentence actually means _"Do you need a taxi?"._

As another example, if you were to ask someone other than Siti the question **_Apakah Siti dari Brunei?_**, it would mean _"Is Siti from Brunei?"_. However, if you were to ask Siti directly it would mean _"Are you from Brunei?"_.
