# Questions (Vocab)

[Questions Grammar](/Grammar/Questions.md)

|                                 |                               |
| ------------------------------- | ----------------------------- |
| Apa, apakah?                    | What?                         |
| Apa itu?                        | What's that?                  |
| Siapa, siapakah?                | Who?                          |
| Siapakah orang itu?             | Who is that person?           |
| Bila, bilakah?                  | When?                         |
| Bila penerbangan ke London?     | When is the flight to London? |
| Bilakah mesyuarat?              | When is the meeting?          |
| Bagaimana...?, Bagaimanakah...? | How's...?                     |
|                                 | How are...?                   |
|                                 | How was...?                   |
|                                 | How were...?                  |
|                                 | What kind of...?              |
|                                 | What's... like?               |
| Bagaimana filem itu?            | How was that film?            |
| Bagaimana emak anda?            | How is your mother?           |
