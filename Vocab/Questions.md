## Where? Di Mana?

|                         |                             |
| ----------------------- | --------------------------- |
| Di mana...?             | Where...?                   |
| Di manakah?             |                             |
| Di mana anda tinggal?   | Where do you live/stay?     |
| Ke mana...?             | Where to..., to where?      |
| Ke manakah?             |                             |
| Ke mana anda pergi?     | Where are you going (to)?   |
| Dari mana...?           | Where from...?, from where? |
| Dari manakah?           |                             |
| Dari mana anda berasal? | Where do you come from?     |

## How Many? Berapa?

|                     |                      |
| ------------------- | -------------------- |
| Berapa...?          | How many...?         |
| Berapa **lama**?    | How long? (time)     |
| Berapa **panjang**? | How long? (distance) |
| Berapa kali?        | How many times?      |
