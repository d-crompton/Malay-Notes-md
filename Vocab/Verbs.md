## Bercakap, Bertutur and Berbual

There are three different ways to say _to speak_ in Malay - **_bercakap_**, **_bertutur_** and **_berbual_**. The difference between **_bercakap_** and the other two is that **_bertutur_** and **_berbual_** must be followed by **_dalam_** (_in_).

|                                                  |                             |
| ------------------------------------------------ | --------------------------- |
| Anda _bercakap_ Bahasa Melayu dengan baik sekali | You speak Malay very well   |
| Mereka _bertutur dalam_ Bahasa Cina dengan fasih | They speak Chinese fluently |
| Bolehkah Devinder _berbual dalam_ Bahasa Melayu  | Can Devinder speak Malay?   |

## Biar and Mari

**_Biar_** means _'to let or to allow,'_ and **_Mari_** means _'let us,'_ or _'let's'_.

**_Mari_** can be seen as an alternative to **_"biar kita"_**. Using **_"biar kita"_** for '_let us_' can be seen as 'wierd'.

Additionally, **_"mari saya bantu"_** (_let me come help_) can be seen as a 'kinder' way of saying **_"biar saya bantu"_** (_let me help_).

|                              |                     |
| ---------------------------- | ------------------- |
| Biar saya bantu              | Let me help         |
| Biar kita pergi / Mari pergi | Let us go/ Let's go |

## Hendak and Nak

**_Nak_** is a shortening of **_Hendak_** meaning _to want_ or _to need_.

Another way of saying _to want_ is **_Mahu_**.

## Ter-

Verbs beginning with **_ter-_** refer to an undesired outcome resulting from something outside of the speaker's control:

|            |             |
| ---------- | ----------- |
| Terlewat   | To be late  |
| Tersangkut | To be stuck |

## Adverbs

|                                               |                                 |
| --------------------------------------------- | ------------------------------- |
| _[verb]_ dengan baik sekali                   | To _[verb]_ very well           |
| _[verb]_ dengan lebih perlahan                | To _[verb]_ more slowly, slower |
| Bolehkah anda bercakap dengan lebih perlahan? | Can you speak slower?           |
