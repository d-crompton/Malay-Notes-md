## Nationalities

When speaking about nationality, you add **_Orang_** (_person_) before the country name.

|                    |                |
| ------------------ | -------------- |
| Orang Amerika      | An American    |
| Saya orang Ireland | I am Irish     |
| Orang Cina         | Chinese person |

Even though _China_, the country, is the same in English and Malay, Chinese people are **_Orang Cina_** in Malay.

## Languages

When speaking about languages, you add **_Bahasa_** (_language_) before the country name.

|                                    |                      |
| ---------------------------------- | -------------------- |
| Bahasa Itali                       | Italian (language)   |
| Dia boleh bercakap Bahasa Perancis | She can speak French |
| Bahasa Inggeris                    | English              |

_England_ (the place) has the same spelling and pronunciation in English and Malay. However, English (the language) is **_Bahasa Inggeris_**.

Similarly, _China_ (the place) is the same but the Chinese language is **_Bahasa Cina_** or **_Bahasa Mandarin_**.
