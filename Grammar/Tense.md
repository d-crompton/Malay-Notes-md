# Tense

Tense is conveyed in Malay by using specific words. For example, **_sudah_** (_already_) and **_sedang_** (_now/currently_) are used to convey past tense and the continuous present tense respectively:

|                                   |                                  |
| --------------------------------- | -------------------------------- |
| Dia _sudah_ menulis surat         | She has already written a letter |
| Kami _sedang_ menonton televisyen | We are currently watching TV     |
| Saya _sedang_ sibuk               | I am currently busy              |
