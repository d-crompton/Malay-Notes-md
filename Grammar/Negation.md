# Negation

Malay uses **_Tidak_** or **_Bukan_** to make a sentence negative (i.e., _"no"_ or _"not"_). But there is a distinction between their usage.

**_Tidak_** is used directly before a **verb** or **adjective** to make it negative.

|                      |                      |
| -------------------- | -------------------- |
| Dia minum            | He drinks            |
| Dia _tidak_ minum    | He _doesn't_ drink   |
| Saya tahu            | I know               |
| Saya _tidak_ tahu    | I _don't_ know       |
| Dia sihat            | He is healthy        |
| Dia _tidak_ sihat    | He _isn't_ healthy   |
| Mereka mabuk         | They are drunk       |
| Mereka _tidak_ mabuk | They _are not_ drunk |

**_Bukan_** is used before a **noun** or **pronoun**

|                             |                              |
| --------------------------- | ---------------------------- |
| Saya orang Malaysia         | I am Malaysian               |
| Saya _bukan_ orang Malaysia | I am _not_ Malaysian         |
| Ini kucing                  | This is a cat                |
| Ini _bukan_ kucing          | This is not a cat            |
| Dia presiden? _Bukan_ dia   | Is he the president? Not him |

**_Tidak_** and **_Bukan_** can also be used as one-word responses to direct questions. However, which one you use depends on the focus of the question.

- **_"Apakah kawan anda penari?"_** means _"Is your friend a dancer?"_. As the focus is the noun **_penari_** (_dancer_) you would reply **_bukan_**.
- **_"Apakah dia pendek"_** means _"Is he short?"_. The focus is the adjective **_pendek_** (_short_), so you would reply with **_tidak_**.
- **_"Apakah dia bekerja di Sabah?"_** means _"Does she work in Sabah?"_. The focus of this is the verb **_bekerja_** (_to work_), therefore you say **_tidak_**.
