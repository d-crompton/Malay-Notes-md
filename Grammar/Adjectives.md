# Adjectives

## Adding Adjectives or Descriptors

In Malay, nouns can be described using adjectives, nouns or verbs. However, unlike English, the describing word is placed _after_ the noun

|                    |                     |
| ------------------ | ------------------- |
| **Adjectives**     |                     |
| Kereta _baru_      | New car             |
| Gambar _menarik_   | Interesting picture |
| **Nouns**          |                     |
| Stesen _bas_       | Bus station         |
| Orang _Wales_      | Welsh person        |
| **Verbs**          |                     |
| Lapangan _terbang_ | Airport, airfield   |
| Bilik _mandi_      | Washroom, bathroom  |

When more than one describing word (including _possessive pronouns_ or _names_) occur in a phrase, the word order is the reverse of what you would expect in English.

|                                 |                         |
| ------------------------------- | ----------------------- |
| Sekolah _Bahasa_ **_Inggeris_** | English Language School |
| Pasport _teman_ **_saya_**      | My friend's passport    |
