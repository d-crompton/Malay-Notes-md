## Direct Questions

A **_direct_** question requires a _yes_ or _no_ answer and an **_indirect_** question requires an explanation. You turn a statement into a **_direct_** question in Malay by adding **_Adakah_** or **_Apakah_**. For example:

|                  |                |
| ---------------- | -------------- |
| Ini buku         | This is a book |
| Adakah ini buku? | Is this a book |

In informal Malay **_Adakah_** and **_Apakah_** may be omitted completely. Instead the statement is spoken with a _rising intonation_ at the end which signals that it is a question. **_Dia penat_** can mean either _"she is tired"_ or _"is she tired?"_ depending on your intonation.

## Bagaimana Dengan...

**_Bagaimana(kah) dengan..._** is a more colloquial way of using _bagaimana_ to ask how something was.

|                                    |                      |
| ---------------------------------- | -------------------- |
| Bagaimana dengan penerbangan anda? | How was your flight? |

# Bolehkah

**_Bolehkah_** can be translated as _could you..._. The **_-kah_** suffix is optional and used colloquially. Malay people tend to use the **_-kah_** when asking questions in a written form.
